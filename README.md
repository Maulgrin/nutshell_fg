# Nutshell FG Starter Ruleset

This is a Fantasy Grounds starter ruleset built on CoreRPG for Nutshell.

Included:
- Nutshell character sheet
- 2d6 + skill roll buttons
- Ranged and close-combat attack buttons
- Strikes / Strain derived automation
- CoreRPG combat tracker support
- Sample NPC templates
- Reference chart story entries

## Install

Drop the `nutshell_fg` folder into your Fantasy Grounds `rulesets` directory.

## Notes

- TN is host-controlled via the desktop TN widgets.
- Attack rolls report strikes from degree of success.
- This is a starter port based on the real Foundry Nutshell rules logic and CoreRPG structure.
